ตรวจอะไรบ้าง

ออกแบบให้ทีมธุรกิจ ทีม Product ทีม Marketing และทีมพัฒนาเว็บเข้าใจผลลัพธ์ได้เร็ว โดยไม่ต้องเริ่มจากศัพท์เทคนิคทั้งหมด

👁️

Visual Access

สี ตัวอักษร contrast ขนาดฟอนต์ และองค์ประกอบที่กระทบผู้ใช้สายตาเลือนราง

⌨️

Keyboard Access

ตรวจว่าผู้ใช้สามารถใช้งานเมนู ปุ่ม ฟอร์ม และ flow สำคัญโดยไม่ต้องใช้เมาส์ได้หรือไม่

🔊

Screen Reader Access

ตรวจโครงสร้าง heading, alt text, label, landmark และชื่อของปุ่มสำหรับผู้ใช้ screen reader

📝

Form & Interaction

ตรวจช่องกรอก error message คำอธิบาย และสถานะของระบบเมื่อเกิดข้อผิดพลาด

🧭

Navigation Structure

ตรวจลำดับเนื้อหา เมนู ลิงก์ breadcrumb และการเดินทางของผู้ใช้ในหน้าเว็บ

💬

Content Clarity

ตรวจความชัดเจนของภาษา คำแนะนำ ป้ายกำกับ และเนื้อหาที่ช่วยให้ผู้ใช้เข้าใจบริการ

Inclusive Ready Score

แปลงผลตรวจเชิงเทคนิคให้เป็นระดับที่ทีมตัดสินใจต่อได้ง่าย เห็นทั้งความเสี่ยง จุดแข็ง และงานที่ควรทำต่อ

0–39

Needs Attention

มีอุปสรรคสำคัญต่อการใช้งาน ควรวางแผนแก้ไขก่อนเผยแพร่หรือก่อนทำแคมเปญใหญ่

40–59

Getting Started

เริ่มมีพื้นฐานบางส่วน แต่ยังต้องตรวจและปรับหลายจุด โดยเฉพาะ flow สำคัญของผู้ใช้

60–79

Almost Accessible

โครงสร้างพอใช้งานได้ แต่ยังมีจุดเสี่ยงที่ควรแก้เพื่อให้เข้าถึงได้ราบรื่นขึ้น

80–100

Inclusive Ready

มีความพร้อมสูง เหมาะต่อยอดเป็น accessibility statement, certification หรือ case study

จากผลตรวจ สู่บริการจริง

เครื่องมือตรวจอัตโนมัติช่วยเปิดประเด็นได้เร็ว แต่การเข้าถึงจริงควรมี human review โดยผู้เชี่ยวชาญและผู้ใช้งานที่หลากหลายร่วมตรวจด้วย

สิ่งที่คุณจะได้รับเมื่อขอรายงาน

  • Accessibility Rating เบื้องต้น
  • รายการ issue ที่ควรแก้ก่อน
  • คำอธิบายผลกระทบต่อผู้ใช้
  • Quick wins ที่ทีมพัฒนาแก้ได้เร็ว
  • ข้อเสนอ next step: audit, training, consulting หรือ remediation

ขอ Human Review จาก Vulcan

ส่งข้อมูลให้ทีม Vulcan Coalition ช่วยประเมินเชิงคุณภาพเพิ่มเติม และติดต่อกลับพร้อมข้อเสนอแนะที่นำไปใช้ต่อได้จริง

  • เหมาะสำหรับเว็บไซต์องค์กร แอปบริการ ลูกค้าภาครัฐ และแคมเปญดิจิทัล
  • ต่อยอดเป็น WCAG Audit, Accessibility Training หรือ UX Review ได้
  • สื่อสารผลลัพธ์เป็นภาษาที่ทีมธุรกิจและทีมเทคนิคเข้าใจร่วมกัน

Prototype นี้ยังไม่ได้ส่งข้อมูลไปยังระบบจริง สามารถเชื่อมต่อกับ Google Form, HubSpot, Make, Zapier หรือ backend ของ Vulcan ได้ภายหลัง

ได้รับคำขอแล้ว ขอบคุณครับ ทีม Vulcan Coalition จะติดต่อกลับทางอีเมล
